개요

이 영상은 윈도우 로컬 계정 비밀번호 분실로 로그인에 어려움을 겪는 사용자를 위해 스마트폰을 활용하여 윈도우 PE 복구 USB를 만들고, 비밀번호를 초기화하는 방법을 상세히 안내합니다. PC 없이 안드로이드 스마트폰과 OTG 어댑터, USB만 있으면 누구나 쉽게 따라 할 수 있도록 단계별로 설명하며, 포맷 없이 데이터를 유지하면서 문제를 해결하는 실용적인 팁을 제공합니다.

내용

  • [문제 제기] 윈도우 암호 변경 후 로그인 불가 상황 (00:00:00)

    • 윈도우 사용 중 암호 변경 후 정확한 비밀번호를 입력했음에도 로그인되지 않는 문제 발생 (00:00:09)
    • 마이크로소프트 계정은 인터넷 연결로 해결 가능하나, 로컬 계정의 경우 해결이 어려움 (00:00:20)

      • ※ 로컬 계정: 컴퓨터 자체에 사용자 정보가 저장되는 계정으로, 인터넷 연결 없이도 로그인할 수 있지만 클라우드 기반 서비스와 연동되지 않음.

    • 기존의 복잡한 해결 방법에 어려움을 느끼는 사용자들을 위한 새로운 방법 제시 (00:00:33)

  • [해결책 소개] 스마트폰으로 PE 복구 USB 만들기 (00:00:43)

    • 스마트폰 하나로 윈도우 PE 복구 USB를 만들어 비밀번호를 간단하게 초기화하는 방법 소개 (00:00:43)

      • ※ PE 복구 USB: Windows Preinstallation Environment (윈도우 사전 설치 환경)가 담긴 USB로, 운영체제 설치 전이나 시스템 복구 등 다양한 진단 및 복구 작업을 수행할 수 있는 최소한의 윈도우 환경을 제공함.

    • 안드로이드 폰에서만 가능하며, 앱 설치, ISO 파일 다운로드, USB 연결만으로 완료 (00:00:55)

      • ※ ISO 파일: CD나 DVD 같은 디스크의 내용을 하나의 파일로 압축하여 저장한 이미지 파일. 운영체제 설치 파일 등에 주로 사용됨.

    • USB 연결을 위한 OTG 어댑터 필요 (00:01:06)

      • ※ OTG 어댑터: On-The-Go의 약자로, 스마트폰이나 태블릿 같은 모바일 기기에 USB 저장 장치나 키보드, 마우스 등을 연결할 수 있게 해주는 변환 젠더.

    • 기존 PC에서 부팅 USB를 만들던 방식보다 훨씬 간편 (00:01:21)

  • [준비물 안내] 필요한 장비 및 파일 (00:01:53)

    • 안드로이드 스마트폰, 8GB 이상 USB, OTG 어댑터, 윈도우 11 PE ISO 이미지 파일 (00:01:53)
    • 윈도우 11 PE ISO 파일은 '홍차의 꿈' 블로그에서 다운로드 가능하며, 링크는 고정 댓글 또는 영상 설명란 참조 (00:02:03)
    • ISO 파일은 구글 드라이브에서 직접 다운로드 (00:02:23)
    • 파일 용량은 약 660MB로, 빠르게 다운로드 가능 (00:02:36)
    • 다운로드 확인은 스마트폰 '내 파일' 앱의 다운로드 폴더에서 가능 (00:02:44)

  • [PE 부팅 USB 생성] EtchDroid 앱 활용 (00:03:02)

    • EtchDroid 앱을 구글 플레이스토어에서 설치 후 실행 (00:03:06)

      • ※ EtchDroid: 안드로이드 스마트폰에서 ISO 또는 IMG 파일을 USB 드라이브에 기록하여 부팅 가능한 USB를 만들 수 있게 해주는 앱.

    • 스마트폰에 USB 연결 후 앱에서 '이미지 쓰기' 버튼 클릭 (00:03:14)
    • 다운로드 받은 윈도우 11 PE ISO 이미지 파일 선택 (00:03:23)
    • "윈도우 저 ISO인가요?" 알림 팝업에 '무시하고 계속' 선택 (00:03:37)
    • 대상 USB 장치 접근 허용 및 쓰기 권한 부여 (00:03:45)
    • '이미지 쓰기' 버튼을 눌러 USB에 PE 파일 기록 시작 (00:04:11)
    • '이미지가 성공적으로 작성되었어요' 메시지가 뜨면 작업 완료 (00:04:44)

  • [PC 부팅 설정] BIOS 진입 및 부팅 순서 변경 (00:04:59)

    • 생성된 부팅 USB를 컴퓨터에 연결 (00:04:59)
    • 컴퓨터 재부팅 시 BIOS 화면으로 진입 (삼성: F2, 다른 브랜드는 인터넷 검색 필요) (00:05:03)

      • ※ BIOS: Basic Input/Output System의 약자로, 컴퓨터의 하드웨어 초기화 및 운영체제 로딩을 담당하는 펌웨어.

    • BIOS 내 '부트 메뉴'의 '부트 프라이올리티'에서 부팅 순서 변경 (00:05:23)
    • '윈도우 부트 매니저' 대신 USB 장치를 1순위 부팅으로 설정 (00:05:35)
    • 설정 저장 후 재부팅하면 윈도우 PE 복구 화면으로 진입 (00:05:46)
    • 윈도우 PE는 일반 윈도우와 유사하며 복구 전용 환경임 (00:06:00)

  • [비밀번호 초기화] NTPWEdit 프로그램 사용 (00:06:08)

    • 윈도우 PE 바탕화면에서 NTPWEdit 아이콘 실행 (00:06:08)

      • ※ NTPWEdit: 윈도우 로컬 계정의 비밀번호를 초기화하거나 변경할 수 있는 경량 프로그램으로, 특히 비밀번호 분실 시 유용하게 사용됨.

    • 프로그램 우측 상단 '열기' 버튼 클릭하여 로컬 사용자 계정 목록 불러오기 (00:06:32)
    • 비밀번호를 분실한 로컬 계정 선택 (00:06:43)
    • '암호 변경' 버튼 클릭 후 새 암호 입력 (비워서 초기화도 가능) (00:06:54)
    • 새 암호 입력 후 '확인' 버튼 클릭 (00:07:07)
    • 마지막으로 '변경 사항 저장' 버튼 클릭 (00:07:11)
    • 버튼이 비활성화되면 저장 완료 (00:07:14)
    • '끝내기' 버튼으로 프로그램 종료 (00:07:23)

  • [최종 확인 및 결론] 로그인 성공 및 마무리 (00:07:31)

    • 모든 작업 완료 후 컴퓨터 재시작 및 USB 제거 (00:07:31)
    • 새로운 비밀번호로 윈도우 로그인 시도 (00:07:39)
    • 정상적으로 로그인되어 윈도우 바탕화면 진입 확인 (00:07:47)
    • 이 방법은 윈도우 11 및 윈도우 10 로컬 계정 비밀번호 분실 시 포맷 없이 해결 가능한 유용한 방법 (00:07:56)

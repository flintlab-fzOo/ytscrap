개요

이 영상은 윈도우 로컬 계정 비밀번호를 분실하여 로그인이 불가능할 때, PC 없이 안드로이드 스마트폰과 USB, OTG 어댑터를 활용하여 윈도우 PE 복구 USB를 만들고, 이를 통해 NTPW 에디트 프로그램을 사용하여 비밀번호를 안전하게 초기화하는 방법을 단계별로 상세히 안내합니다. 특히 윈도우 11 및 윈도우 10 로컬 계정 비밀번호 분실 문제 해결에 유용하며, 포맷 없이 데이터를 보존하며 시스템 사용을 원활하게 이어갈 수 있도록 돕습니다.

내용

[서론] 윈도우 비밀번호 분실 문제 및 해결책 소개 (00:00:00)

  • 윈도우 사용 중 암호 변경 후 로그인이 안 되는 난감한 상황에 대한 언급.
  • 마이크로소프트 계정과 달리 로컬 계정 비밀번호 분실 시의 막막함과 기존 해결 방법의 어려움.
  • 포맷 없이 스마트폰 하나로 윈도우 PE 복구 USB를 만들어 비밀번호를 간단하게 초기화하는 방법을 소개.
  • 안드로이드 폰만 가능하며, OTG 어댑터가 필요하며 다이소 등에서 쉽게 구할 수 있음.
  • 기존 컴퓨터를 이용한 루퍼스 방식보다 스마트폰 방식이 훨씬 간편함을 강조.

※ 로컬 계정: 인터넷에 연결되지 않고 특정 컴퓨터에만 사용자 정보가 저장되는 계정.

※ 마이크로소프트 계정: 인터넷에 연결되어 마이크로소프트 서비스(OneDrive, Outlook 등)와 연동되는 계정.

※ PE: Preinstallation Environment의 약자로, 윈도우 설치 또는 복구 작업을 위해 사용하는 최소화된 윈도우 환경.

※ OTG 어댑터: On-The-Go의 약자로, 스마트폰과 같은 모바일 기기에 USB 장치(USB 메모리, 키보드 등)를 직접 연결할 수 있게 해주는 변환 젠더.

※ 루퍼스: USB 부팅 디스크를 만들 때 사용되는 무료 오픈소스 유틸리티.

[준비물] 필요한 도구 및 파일 안내 (00:01:53)

  • 안드로이드 스마트폰, 8GB 이상의 USB, OTG 어댑터, 윈도우 11 PE ISO 이미지 파일이 필요함.

※ ISO 파일: CD, DVD 또는 블루레이 디스크의 내용을 하나의 파일로 압축해 놓은 디스크 이미지 파일.

[PE ISO 파일 다운로드] 윈도우 11 PE ISO 파일 구하기 (00:02:03)

  • 홍차의 꿈 블로그에서 윈도우 11 PE ISO 이미지 파일을 다운로드하는 방법 안내.
  • 영상 설명란의 상세정보 링크를 통해 해당 사이트에 접속하여 첫 번째 링크를 클릭, 구글 드라이브에서 ISO 파일을 다운로드함.
  • ISO 파일 용량은 약 660MB로 크지 않아 빠르게 다운로드 가능.
  • 다운로드된 파일은 스마트폰의 다운로드 폴더(내 파일 앱)에서 확인 가능.

[부팅 USB 생성 (Android)] 엣치드로이드 앱을 사용하여 USB 만들기 (00:03:02)

  • 스마트폰으로 윈도우 11 PE 부팅 USB를 만들기 위해 '엣치드로이드(etchdroid)' 앱 설치 필요.
  • 구글 플레이스토어에서 앱을 설치하고 실행.
  • 스마트폰에 USB를 연결한 후 앱에서 '이미지 쓰기' 버튼 클릭.
  • 다운로드받은 윈도우 11 PE ISO 이미지 파일 선택.
  • '윈도우 ISO인가요?' 알림 팝업 시 '무시하고 계속' 클릭.
  • 대상 USB 장치 항목에서 '접근 허용' 버튼을 탭하여 쓰기 권한 부여.
  • 권한 허용 후 '이미지 쓰기' 버튼을 눌러 USB에 ISO 파일 기록 시작.
  • 작업 완료 시 '이미지가 성공적으로 작성되었어요' 메시지 확인.

[바이오스 부팅 설정] USB로 컴퓨터 부팅 순서 변경 (00:05:03)

  • 생성된 부팅 USB를 컴퓨터에 연결.
  • 컴퓨터 재부팅 시 바이오스 진입 키 (삼성 컴퓨터는 F2, 다른 브랜드는 검색 후 확인)를 연타하여 바이오스 화면 진입.
  • 바이오스 화면에서 'Boot' 메뉴의 'Boot Priority' 항목을 찾아 진입.
  • 'Boot Option #1' 항목을 USB 장치로 변경하여 최우선 부팅 순서로 설정.
  • 설정 저장 후 재부팅하면 윈도우 PE 복구 화면으로 진입.

※ 바이오스: Basic Input/Output System의 약자로, 컴퓨터의 하드웨어를 초기화하고 운영체제를 부팅하는 기본적인 펌웨어.

[PE 환경 진입 및 NTPW 에디트 실행] 윈도우 PE 환경 활용 (00:05:56)

  • 윈도우 PE는 일반 윈도우 환경과 유사하며 비밀번호 초기화 등 복구 작업에 사용되는 전용 환경.
  • PE 바탕화면에서 'NTPW 에디트' 아이콘을 찾아 실행.
  • 이 프로그램은 로컬 계정 비밀번호 초기화 및 변경에 효과적인 경량 프로그램임.

※ NTPW 에디트: Windows NT 계열 운영체제(Windows XP, Vista, 7, 8, 10, 11 등)의 로컬 사용자 계정 비밀번호를 변경하거나 초기화할 수 있는 유틸리티.

[비밀번호 초기화/변경] NTPW 에디트를 통한 비밀번호 재설정 (00:06:32)

  • NTPW 에디트 프로그램 실행 후 '열기' 버튼을 클릭하여 사용자 계정 정보를 불러옴.
  • 표시된 로컬 사용자 계정 목록 중 비밀번호를 분실한 계정 선택.
  • 하단의 '암호 변경' 버튼 클릭.
  • '새 암호'와 '암호 확인' 필드에 새로운 비밀번호를 동일하게 입력하거나 비워 초기화.
  • '확인' 버튼 클릭 후 마지막으로 '변경 사항 저장' 버튼 클릭.
  • '변경 사항 저장' 버튼이 비활성화되면 내용이 정상적으로 저장된 것임.
  • 우측 하단의 '끝내기' 버튼을 눌러 프로그램 종료.

[최종 확인 및 마무리] 윈도우 로그인 확인 (00:07:31)

  • 모든 작업 완료 후 컴퓨터를 다시 시작하고 USB 장치 제거.
  • 윈도우 로그인 화면에서 방금 변경한 새로운 비밀번호를 입력하여 로그인 시도.
  • 정상적으로 암호가 적용되었다면 로그인 오류 없이 윈도우 바탕 화면으로 진입 가능.
  • 비밀번호 분실로 막혔던 시스템 사용을 다시 원활하게 할 수 있음.

[결론] 방법 요약 및 적용 범위 (00:07:56)

  • 윈도우 11 로컬 계정 비밀번호 분실 시 포맷이나 초기화 대신 PE 복구 USB를 활용하는 간단한 방법으로 안전하게 초기화 및 복구 가능.
  • 스마트폰을 이용한 윈도우 11 PE 복구 USB 제작부터 바이오스 설정, NTPW 에디트 사용, 바탕 화면 진입까지의 전 과정을 자세히 설명.
  • 윈도우 10 버전도 동일한 방법으로 적용 가능.
  • IT 지식이 많지 않은 사용자도 단계별로 쉽게 따라 할 수 있을 정도로 직관적임.
